# Casa Hidalgo

Sistema de reservas — Centro Histórico de Querétaro.
Stack: HTML+JS · Supabase · MercadoPago · Netlify.

## Setup local
1. cp public/config.example.js public/config.js
2. Edita public/config.js con tus credenciales
3. npx netlify dev (abre http://localhost:8888)

## Deploy
Push a main → Netlify hace deploy automático.
Variables de entorno: configurar en Netlify Dashboard.
